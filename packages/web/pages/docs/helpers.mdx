# Helpers

`Phake` supports all [Handlebars built in helpers](https://handlebarsjs.com/guide/builtin-helpers.html) but it also ships some convenient helpers to help with
data generation. 

## faker
The faker helper is used to access the FakerJS library. It calls the fakerjs library function and passes on the supplied arguments to the fakerJS function
```template
{{faker 'faker.function' }}
```

#### Examples
Faker with no parameters 
```template
{{faker 'string.alpha' 10}}
```


Faker with simple parameters
```template
{{faker 'person.firstName'}}
```

Faker with named parameters
```template
{{faker 'date.birthdate' min=18 max=65 mode='age'}}
```

## repeat

The repeat helper is useful for repeating a partial or piece of template code serveral times. It generates a comma
separated list, thus great for generating arrays

```template
{{#repeat number}}
    template-code-goes-here
{{/repeat}}
```

#### Example
This will generate an array of 5 integers
```template
[
    {{#repeat 5}}
        {{faker 'number.int'}}
    {{/repeat}}
]
```

## randomize
This helper randomizes a list of parameters passed to it

```template
{{randomize param1 param2  ...}}
```

#### Example
```template
{{randomize 'main' 'master'}}
```
